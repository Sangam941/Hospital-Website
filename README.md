A modern, responsive, and feature-rich hospital web application built with **React**, **TypeScript**, and **Tailwind CSS**. This project demonstrates a clean UI/UX designed for healthcare providers, featuring doctor profiles, service listings, and interactive contact systems.

## ‚ú® Features

- **Dynamic Hero Sections**: Visually stunning headers with background images and glassmorphism.
- **Service Exploration**: Categorized medical services with search functionality.
- **Doctor Directory**: Reusable `DoctorCard` components showcasing specialists, ratings, and experience.
- **Interactive Contact Form**: State-managed form with real-time feedback and validation.
- **Full-Width Map Integration**: Integrated Google Maps for easy location tracking.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **Patient Testimonials**: "Voices of Recovery" carousel component for social proof.

## üöÄ Getting Started

Follow these steps to set up the project locally:

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sangam941/Hospital-Website.git
   cd hospital-website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Visit the app**:
   Open [http://localhost:5173](http://localhost:5173) in your browser.

- `public/`: Assets like images and icons.
